# **App Name**: Krishi Mitra

## Core Features:

- User Onboarding: Collect user data such as Phone Number, Location, Crop Type, Farming Methods, and preferred Language upon registration using Firestore.
- Visual Health Dashboard: Implement a dashboard with visual indicators (traffic light system) for overall crop health status.
- Real-time Advisory: Provide personalized, weather-integrated advice (irrigation, fertilizer, harvest) generated by the AI Advisory Engine.
- Smart Photo Analysis: Allow farmers to upload crop images for AI/ML-based pest/disease detection and treatment recommendations tool.
- Market Linkage: Display real-time market prices for registered crops with price comparisons to optimize selling time.
- Action Feedback Loop: Enable farmers to provide feedback on the received advisory to facilitate continuous model improvement using Firestore.
- Admin Dashboard: Dashboard for monitoring system health, data integrations, and AI/ML model performance. Manage farmer accounts and track feedback/adoption rates using Firestore.

## Style Guidelines:

- Background color: Desaturated light green (#E8F5E9) to create a calming and natural feel.
- Primary color: A vibrant green (#4CAF50) to signify growth, health, and prosperity.
- Accent color: A warm yellow (#FFC107) to highlight key actions and important information.
- Headline font: 'Poppins' (sans-serif) for a modern and accessible feel.
- Body font: 'PT Sans' (sans-serif) which pairs well with Poppins for readability.
- Use clear and intuitive icons to represent different actions, crops, and conditions.
- Use subtle animations to provide feedback, guide the user, and enhance the experience.
